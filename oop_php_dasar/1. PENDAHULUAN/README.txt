Procedural Programming :
========================
1. Instruksi dilakukan langkah demi langkah
2. Memecah program menjadi bagian-bagian kecil
3. Disebut prosedur, subroutine atau function
4. Linear / top-to-bottom

Kelebihan :
===========
1. To the point
2. Simplicity & kemudahan implementasi (untuk compiler & interpreter)
3. Mudah ditelusuri
4. Membutuhkan lebih sedikit memory (dibandingkan dengan OOP)

================================================

Object Oriented Programming :
=============================
1. Menyususun semuda kode program dan struktur data sebagai objek
2. Objek adalah unit dasar dari program
3. Objek menyimpan data dan perilaku
4. Objek bisa saling berinteraksi

Kelebihan OOP :
===============
1. Representasi dunia nyata
2. Enkapsulasi & abstraksi data
3. Reusability
4. Skalabilitas & ekstensibilitas
5. Kemudahan pengelolaan
6. Kolabarasi
7. Digunakan oleh Framework

Konsep Basic :
==============
1. Class & Object
2. Property & Method
3. Constructor
4. Object Type
5. Inheritance
6. Visibility / Access Modifier
7. Setter & Getter
8. Static Method

Konsep Advance :
================
1. Abstract & Inference
2. Interceptor
3. Object Cloning
4. Callback & Closures
5. Namespaces & Autoloading
